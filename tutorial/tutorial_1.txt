基本
sample_1.py
from flask import Flask
でFlaskが使えるようになる。
app = Flask(__name__)でなんかインスタンス作ってる。
@app.route("ここにアドレス")
def 関数名():
    return なんか戻り値
ってやるとブラウザ上に戻り値が表示されるっていうのが基本。
アドレスは http://127.0.0.1:5000/～～～みたいなやつ。
なんか引数渡すには、変数名をdummyとして、
("/<dummy>")
みたいにして関数の方でfunc(dummy)とかしてdummyが使える。
他にも渡し方があるのか、ないのか。
関数とかはimportでローカルにいくつも用意してあっち行ったりこっち行ったり。
メインの関数には最後に
if __name__ == "__main__":
    app.run(debug = True)
ってやるとそれが書かれたファイルを実行した時にサーバが立ち上がってテストできる。
